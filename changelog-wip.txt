BotC v1.6.0 - April XX, 2026

Changes and features:
- Added the ability to set your preferred name with /nick. Non-vanilla UI will still display usernames.
- Added a radial menu (opened with C by default) to quickly select nominator and nominee without having to open the Grimoire.
- Added the ability to import a script from a file.
- Added icons to "Teleport to seats" and "Teleport to homes" buttons.
- Added an explanatory tooltip to the "Teleport to homes" button.
- Added a "Special Actions" menu to the Grimoire. This menu contains several character-specific buttons:
  - Toggle Organ Grinder ability
  - Toggle Voudon ability
  - Toggle Banshee ability
  - Announce Banshee
  - Start cult vote
  - Restore ghost votes
  - Announce Al-Hadikhia
  - Announce Leviathan
  - Announce Fearmonger
  - Play Roshambo
  - Start Boomdandy timer
- Added timer settings:
  - TP players to seats
  - Pause during nominations
- Players can now whisper to their neighbors and to Storytellers during the game using /whisper. When a game is not active, anyone can whisper to anyone. These messages are always visible to Storytellers.
- Many texts are now translation strings. This is not yet comprehensive.
- Voice chat management now uses advancements instead of block detection so as to allow for vertical and float-based sizing.
- Many commands now check that a game is active before they can be run.
- /tell (and its aliases) are now disabled for non-Storytellers.
- Many commands that are not relevant to the game are hidden for non-ops.
- Storytellers no longer need to be opped in order to Storytell. An operator will still need to be the one to add or remove a Storyteller.

Technical changes:
- Character string IDs are now the same as TPI's (i.e. pit_hag is now pithag). Resource packs containing character icons will need to be updated.
- Character numerical IDs are now in alphabetical order and categorized by character type. The alphabetical order will most likely not be maintained when new characters are added, they'll just be tacked on the end.
  - 001-199: Townsfolk
  - 200-299: Outsiders
  - 300-399: Minions
  - 400-499: Demons
  - 500-599: Travellers
  - 600-699: NPCs
- Storyteller permission checks to run commands are now within the functions instead of the command json file.
- TAB mod has been removed in favor of CustomNameTags, which is much less buggy and lets pronouns be seen at a greater distance.
- Script data is no longer stored on players, and is now exclusively stored and referenced in ct:script

Fixed bugs:
- #70: Block sounds are not muted on nights 1 and 3.
- #80: Starting a game with more than 15 non-ST players in the lobby breaks the game.
- #87: Compasses not handed out on night one.
- #96: Player Names are incorrectly formatted in the Al-Had announcement due to Markdown. (thanks @Jvkeh!)
- #97: Non-Storytellers using the /grimoire command open the Storyteller Grim.
- #99: Spectators get given Village Idiot/Legion tokens. (thanks @Jvkeh!)
- Spectator 👁️ in voice chats icon uses the wrong font.
- The button to change seat 15's character is clickable when there are not 15 players.
- Custom banner patterns refer to a nonexistent translation string.
- There are several accessible holes in the terrain.
- Some voice chats can be entered or left by standing in non-whole blocks (panes, iron bars, fences).
- The Acrobat is missing a * in its character description.
- "Dev Mode" does not persist across /reloads.
- Players dying during the day doesn't update alive player/dead votes count in the Grimoire.
